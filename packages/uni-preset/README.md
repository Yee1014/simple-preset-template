## 开发准备

- 项目支持 ts，微信开发者工具不需要打开 `JS编译成ES5`
- 项目格式化使用 `prettier`
- 项目包管理工具使用 `pnpm`
- Node 版本需要大于等于 `14.18.0`

## 支持平台

- 微信小程序
